Краткая справка

Для записи на секцию введите критерии поиска в электронную форму и нажмите кнопку "Применить".
Поиск групп ведётся с учётом индивидуальной группы здоровья и наличия свободных мест (полностью сформированные группы не
        выводятся). При наличии доступных для записи групп выберите наиболее удобный вариант, подтвердите свой выбор с помощью
        кнопки "Записаться".

В случае, если поиск не дал результатов, будут предложены автоматические рекомендации, наиболее близкие Вашим предпочтениям.

Внимание: убедитесь, что вводимые дни и часы занятий по физкультуре не совпадают с Вашим основным расписанием!
Удаление вариантов с идентичным расписанием происходит автоматически!
